# AI Ethics Assignment – Designing Responsible and Fair AI Systems

## Overview
This repository contains all deliverables for the AI Ethics assignment.  
Theme: **Designing Responsible and Fair AI Systems**  
Objective: Evaluate ethical principles, identify and mitigate bias, and apply frameworks to real-world AI scenarios.

---

## Contents
- `answers.pdf` → Written responses (theoretical understanding, case studies, ethical reflection)
- `compas_audit.ipynb` → Jupyter Notebook with COMPAS dataset fairness audit using IBM’s AI Fairness 360
- `report/compas_report.pdf` → 300-word audit summary
- `healthcare_guideline.pdf` → Bonus task: Ethical AI guideline for healthcare
- `report/figures/` → Visualizations generated by notebook
- `requirements.txt` → Python dependencies
- `README.md` → Project summary and instructions

---

## Assignment Parts
1. **Theoretical Understanding (30%)**
   - Algorithmic bias definition + examples
   - Transparency vs. explainability
   - GDPR’s impact on AI
   - Ethical principles matching

2. **Case Study Analysis (40%)**
   - Amazon’s biased hiring tool
   - Facial recognition in policing

3. **Practical Audit (25%)**
   - COMPAS dataset bias audit
   - Visualizations of disparities
   - 300-word report on findings + remediation

4. **Ethical Reflection (5%)**
   - Reflection on personal project and ethical principles

5. **Bonus Task (+10%)**
   - 1-page guideline for ethical AI in healthcare

---

## Tools & Resources
- **Libraries:** [AI Fairness 360](https://github.com/IBM/AIF360), Pandas, Matplotlib, scikit-learn
- **Datasets:** COMPAS Recidivism Dataset, ProPublica’s analysis
- **Frameworks:** EU Ethics Guidelines for Trustworthy AI
